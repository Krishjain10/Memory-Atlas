# Memory Atlas 🗺️

Memory Atlas is a beautifully designed, map-based digital scrapbook app built for iOS and iPadOS using SwiftUI, MapKit, and PencilKit. Pin your favorite locations on an interactive map and create rich, freeform scrapbook pages for each memory!

## 🌟 Features

- **Interactive Map:** Track your location and drop memory pins anywhere on the globe.
- **Digital Scrapbooking:** Create freeform scrapbook pages with full gesture support (drag, rotate, pinch-to-zoom).
- **Rich Elements:** Enhance your pages with:
  - 🖼️ **Photos:** Import pictures from your photo library.
  - 📝 **Text & Sticky Notes:** Add customizable text boxes and colorful sticky notes.
  - ⭐️ **Shapes & Stickers:** Place vector shapes and emoji stickers.
  - 🔖 **Washi Tape & Stamps:** Decorate with various washi tape patterns and travel-themed stamps (passports, boarding passes, etc.).
- **PencilKit Integration:** Freehand draw directly on your scrapbook pages using Apple's native drawing tools.
- **Layering System:** Full support for layering elements (bring to front, send to back) and locking them in place.

## 🛠️ Technologies Used

- **SwiftUI:** For a modern, declarative user interface.
- **MapKit:** For the interactive map and pins.
- **PencilKit:** For the native drawing canvas experience.
- **CoreLocation:** To determine and track the user's current location.

## 📱 Requirements

- **Platform:** iOS 17.0+ / iPadOS 17.0+
- **IDE:** Xcode or Swift Playgrounds

## 🚀 Getting Started

1. Clone this repository: 
   ```bash
   git clone https://github.com/Krishjain10/Memory-Atlas.git
   ```
2. Open the `.swiftpm` project folder in Xcode or Swift Playgrounds.
3. Build and run on an iOS 17+ simulator or physical device.
